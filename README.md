# Orion-finance-transformation
This repository documents a multi-year Finance Transformation programme at Orion Industrial Technologies B.V., a fictional €6.2B AI-enabled industrial technology company.

The objective of this project is to simulate real-world finance transformation work including:

- Month-end close optimisation
- FP&A transformation
- Financial automation
- Data engineering for finance
- Treasury and cash forecasting
- Pricing and profitability analytics
- M&A and strategic finance

## Structure

- 00_orion_bible → Company foundation and financial architecture
- 01_episode_1 → Month-end close transformation
- 02_episode_2 → Reconciliation automation
- 03_episode_3 → Reporting automation
